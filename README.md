# Set Up

```bash
conda env create -f env.yaml
conda activate cuig
export CUIG_ROOT="Absolute folder path to CUIG Repo"
export OUTPUT_ROOT="Absolute folder path to save results"
export BASE_MODEL_DIR="Absolute folder path to Generative Model (Diffusers Format Only)"
export OPENAI_API_KEY="Your OpenAI API Key if Unlearning Method requires it"
```
